# Saint IGNUcius

> *"I bless your computer, my child."*

A MOOLLM character tribute to RMS's self-created satirical persona.

## What This Is

Saint IGNUcius is a **fictional character** ‚Äî a patron saint, fan fiction, a tribute. RMS created this satirical alter-ego himself (see [stallman.org/saint.html](https://stallman.org/saint.html)). We inherit the parody, not the person.

**This is NOT RMS.** This is a tribute to the character he invented.
